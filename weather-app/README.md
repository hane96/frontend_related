# Weather-App

這是一個用 React 實作的簡易天氣查詢小程式。
主要練習 React Hook、API 串接和狀態管理。

---

## 功能

* 輸入城市名稱或使用定位，查詢即時天氣
* 顯示溫度（°C）、天氣描述
* 查詢歷史紀錄會記錄最近的天氣資料，避免重複加入
* 可以重置查詢資料、歷史紀錄

---

## 使用工具

* HTML/CSS
* React (useState、useEffect、useRef 和自訂hook)
* OpenWeatherMap API
